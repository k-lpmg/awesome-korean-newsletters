# Awesome Korean Newslettters

📰 한국어로 된 분아별 뉴스레터를 소개합니다. 

많은 분들의 참여로 좋은 뉴스레터를 다양한 사람들에게 소개할 수 있었으면 합니다.

> 가나다순으로 작성되어 있습니다.
